## apriltag_ros（christianrauch版）をROS 2 Humbleでビルドして動かすまで

### 1. apriltag_ros とapriltag_msgをクローン

```bash
$ cd ~/worksp/camera_ws/src
$ git clone https://github.com/christianrauch/apriltag_ros.git
$ git clone https://github.com/christianrauch/apriltag_msgs.git
```

---
### 2. 依存解決（rosdep）
```bash
$ cd ~/worksp/camera_ws
$ source /opt/ros/humble/setup.bash
$ rosdep install --from-paths src --ignore-src -r -y
$ colcon build --symlink-install --packages-select apriltag_msgs apriltag_ros
```

### 3. source
```bash
$ source install/setup.bash
```

## カメラキャリブレーション
### 1. install 
```bash
$ sudo apt install -y ros-humble-camera-calibration
```
### 2. v4l2実行後にcalibration実行
```
$ ros2 run v4l2_camera v4l2_camera_node --ros-args -r __ns:=/c920 -r __node:=c920_cam -p video_device:="/dev/video0"
$ ros2 run camera_calibration cameracalibrator --size 9x6 --square 0.025 --ros-args --remap image:=/c920/image_raw --remap camera:=/c920


```
