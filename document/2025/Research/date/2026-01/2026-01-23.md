以下の順で起動してください（初回はビルドも）。

  ビルド（初回/変更後）

  cd /home/hayashi/worksp/camera_ws
  colcon build
  cd /home/hayashi/worksp/gripper_ws
  colcon build

  ターミナル共通（各端末で）

  source /opt/ros/$ROS_DISTRO/setup.bash
  source /home/hayashi/worksp/camera_ws/install/setup.bash
  source /home/hayashi/worksp/gripper_ws/install/setup.bash

  起動（4ターミナル推奨）

  1. GelSight

  ros2 run grasp_everything stream_ros2

  2. Gripper

  ros2 run gripper_control gripper_control

  3. RealSense（必要なら）

  ros2 launch realsense2_camera rs_launch.py enable_color:=true
  enable_depth:=false rgb_camera.color_profile:=640x480x30

  4. 収集オーケストレータ

  ros2 run grasp_everything data_collection_orchestrator --ros-args
  -p object_id:=S1 -p condition:=A -p delta_step_level:=weak

  必要ならこの値を変えてください: object_id, condition,
  delta_step_level, disturbance, hold_sec, disturb_sec など。
