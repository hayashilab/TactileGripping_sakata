## グリッパー動作確認用コマンド
### 1. terminal1
```bash
$ source /opt/ros/humble/setup.bash
$ cd worksp/gripper_ws
$ source install/setup.bash
$ ros2 run gripper_control gripper_control
```
### 2. terminal2
```
$ source /opt/ros/humble/setup.bash
$ cd worksp/gripper_ws
$ source install/setup.bash
$ ros2 service call /gripper/set_zero std_srvs/srv/Empty {}
```
