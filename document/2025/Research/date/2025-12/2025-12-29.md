# 基本コマンド(rpi側)
1. rpiに接続
```bash
$ ssh hayashi-rpi@imaging.local -i hayashi -p 65535
```
2. カメラサーバーとLEDの起動
```bash
$ bash imaging_driver/bash_scripts/setup_cam_and_led.sh
```

**bashの修正後は`chmod +x`で権限を付与しておくこと！
