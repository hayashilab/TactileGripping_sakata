# macvlan
## メリット
物理LAN上の固定IPを付与
コンテナ自体をLAN上の1台として見せることができる->　ラズパイから直接送信可能

## 1. 確認
```bash
$ ip -br link
lo               UNKNOWN        00:00:00:00:00:00 <LOOPBACK,UP,LOWER_UP> 
enp59s0          DOWN           04:bf:1b:55:07:2b <NO-CARRIER,BROADCAST,MULTICAST,UP> 
wlp0s20f3        UP             00:a5:54:56:70:b1 <BROADCAST,MULTICAST,UP,LOWER_UP> 
docker0          DOWN           f2:c4:2d:1f:83:a2 <NO-CARRIER,BROADCAST,MULTICAST,UP> 
```
## 2. 読み方
lo -> ループバック
enp59s0 -> Ethernet
wlp0s20f3 -> Wifi
docker0 -> dockerブリッジ
