Dell G15 5520(NVIDIA RTX 3070 Laptop)
## 1. Download Ubuntu ISO file
Go to [ubuntu](https://ubuntu.com/download/alternative-downloads) and get ISO file you want to install

## 2. Download Rufus and Create Install media
1. Go to [Rufus](https://rufus.ie/ja/)
2. Execute Rufus
3. Select USB device
4. Select ISO file you want to install
5. Push "Start"
6. Close

Depending on the USB device, installation may not be possible after this.

## 3. Install Ubuntu
If you cant install it, redo 2
